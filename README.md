# RASPBERRY_PI_SMS_EXCHANGE
## Feature
- Receive and update SMS content on web page.
- Send SMS to Phone number via web.
- Management all received SMS.

Link demo: https://youtu.be/GVYldJEaf5g

## Hardware
- Raspberry Pi (Wifi/LAN support).
- Module SIM 900.
